📊 Manual Testing Guide

Manual Test Steps:

  1. File Upload Test:
    • Try uploading the sample_data.csv file
    • Verify drag & drop functionality works
    • Check error handling with invalid files
  2. Dashboard Display Test:
    • Verify statistics panel shows correct numbers
    • Check that the map displays and departments change color
    • Confirm top 10 postal codes are listed correctly
  3. Interaction Test:
    • Use the search functionality to filter postal codes
    • Click on top postal codes to highlight departments
    • Hover over map departments to see tooltips
  4. Responsive Design Test:
    • Resize browser window to test mobile view
    • Check that all elements remain accessible
